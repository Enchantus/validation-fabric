# Validation Fabric

Validation Fabric is a risk-aware validation and exact-candidate admission system for modern repositories.

This repository is under active bootstrap. Track the public extraction in [issue #1](https://github.com/Enchantus/validation-fabric/issues/1).

## Provenance

The project was extracted from the Validation Fabric implementation in [Enchantus/SessionBuddie](https://github.com/Enchantus/SessionBuddie). Product-specific configuration and private operational details are intentionally excluded.
